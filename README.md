# nvim-config
Ready to use configuration for nvim.

## Usage
This repository is made to be simply cloned in your `~/.config`.

If you already have an nvim configuration that you want to backup:
```
 mv ~/.config/nvim ~/.config/nvim.bak
```
Then, clone the repository
```
 git clone git@github.com:arscisca/nvim-config.git ~/.config/nvim
```
Enjoy!
